Knight MAPS

Campus Navigation & Planning App (Prototype)

Overview

Knight MAPS is a conceptual campus navigation and planning application designed to help University of Central Florida students quickly locate buildings, view class schedules, and navigate campus efficiently. The project focused on creating an intuitive, low-friction user experience for everyday campus use and time-sensitive class transitions.

Images

Process

The application was prototyped in Figma and evaluated through a structured usability study involving six participants. Each participant completed fifteen core tasks using a think-aloud protocol, which helped uncover usability issues based on real user behavior.

Key features explored include interactive campus maps, building search, step-by-step navigation, and a crowdsourced tips system where students can share location-specific advice.

Results & Key Findings

  • All participants successfully completed all tasks, indicating strong learnability and usability.
  • Navigation-related tasks were generally completed quickly, suggesting clear structure and hierarchy.
  • The primary usability challenges occurred in the crowdsourced tips feature, where users occasionally confused saved tips, personal posts, and inbox-style views.

These findings informed proposed improvements such as clearer labeling and grouping for the tips system, more consistent iconography, and better visibility for step-by-step navigation entry points.

Tools & Methods

Figma (interactive prototyping), usability testing, think-aloud protocol, task-based evaluation, qualitative usability analysis, iterative design refinement, user-centered design principles

Skills Gained

  • User-centered design: Translating user needs into validated design decisions
  • Usability evaluation: Conducting task-based testing using a think-aloud protocol
  • Interaction design: Designing clear navigation flows and visual hierarchy
  • Qualitative analysis: Interpreting user feedback and observed behavior
  • Iterative improvement: Refining designs based on testing results
