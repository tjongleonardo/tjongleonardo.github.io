# CIS4004 – [Web-Based Information Technology]
**University of Central Florida**
## Overview
This repository contains coursework, projects, and assignments completed for CIS4004 – Web-Based Information Technology at the University of Central Florida.
The course focused on front-end web development fundamentals, including:
- HTML5 semantic structure
- CSS styling and layout techniques
- JavaScript for interactivity and DOM manipulation
- Basic responsive design principles
- File organization and version control using Git
  
While these projects were completed for academic credit, they demonstrate practical, hands-on technical skills relevant to entry-level IT and web-focused roles.
